# 🩺 Diyabet Risk Tahmini (Diabetes Prediction Challenge)

Klinik ve biyometrik veriler kullanarak bireylerin
diyabet riskini tahmin eden makine öğrenmesi projesi.

---

## 📌 Genel Bakış

Bu projede sağlık göstergeleri analiz edilerek bireylerin
diyabet hastası olup olmadığı tahmin edilmektedir.
Erken teşhis için klinik karar destek sistemlerine katkı sağlamayı hedefler.

Temel hedefler:
- Sağlık verilerindeki eksiklikleri uygun şekilde doldurmak
- Önemli risk faktörlerini belirlemek (Glucose, BMI vb.)
- Dengesiz veriyle başa çıkarak F1-Score'u maksimize etmek
